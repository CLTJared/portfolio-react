# Installation
```bash
git clone git@github.com:CLTJared/portfolio-react.git
```

## Usage
After cloning locally, run the following commands in your terminal.
1. cd path/to/project-folder
2. npm install
3. npm run start
4. Open a browser to `http://localhost:3000/` to view
